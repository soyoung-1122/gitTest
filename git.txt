이전 버전으로 돌아가기 reset / revert 의 차이  https://victorydntmd.tistory.com/79

-reset은 되돌아간 커밋 버전 이후의 버전은 다 날려버림 
모드 설정에 따라 복구 정도 설정 가능함(기본 mixed)
명령어 : git reset --hard HEAD~   (~ 하나면 바로 전단계 커밋)
git reset --hard ORIG_HEAD   :   실수로 리셋했을 경우

-revert는 살아있음

pull 하면 가져와서 자동 merge됨
fetch 사용하면 합치지 않고 최신 커밋 이력 가져오기만 함


--amend 이전 커밋했던 내용에 추가나 설명 수정 가능
사용 : git commit --amend

git cherry-pick <commit number?id?> 하고 커밋해주면 다른 브랜치에서 특정 커밋만 가져오기 가능.
